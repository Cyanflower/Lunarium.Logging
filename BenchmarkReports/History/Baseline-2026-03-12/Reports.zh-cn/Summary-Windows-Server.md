# 图例说明 (Legends)

- **Mean**        : 所有测量值的算术平均值
- **Error**       : 99.9% 置信区间的一半
- **StdDev**      : 所有测量值的标准差
- **Ratio**       : 相对比率分布的平均值 ([当前值]/[基准线])
- **Gen0**        : 每 1000 次操作触发的 GC 0 代回收次数
- **Allocated**   : 单词操作分配的内存（仅托管内存，包含子调用，1KB = 1024B）
- **Alloc Ratio** : 分配内存比率分布 ([当前值]/[基准线])
- **1 ns**        : 1 纳秒 (0.000000001 秒)
